Flow

How it plays

  1. Reachy draws one move from a shuffled repertoire.
  2. The move repeats three times for the audience.
  3. The camera watches for a thumbs up or thumbs down verdict.
  4. Reachy reacts and quickly returns to the next round.

Run

Use Reachy Mini Control

  1. Open Reachy Mini Control on the machine connected to Reachy.
  2. Install the app from this Hugging Face Space.
  3. Launch it from Control and use the built-in dashboard.
  4. Use it in simulation or on the physical Reachy
